Start the Game: Press any button to start the game

Watch Simon: The game will light up one color (e.g., "Red") and play a sound.

Your Turn: You must copy Simon by pressing that same color button ("Red").

Watch Simon (Round 2): Simon will repeat the first color and add a new one (e.g., "Red," then "Blue").

Your Turn (Round 2): You must repeat the entire sequence in order ("Red," then "Blue").

Keep Going: Simon will keep adding one new color to the end of the sequence each round. You must copy the entire pattern from the beginning every time.

Don't Be Slow: The game gets faster as the sequence gets longer.
